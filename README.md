
## Overview
AgroConnect is an AI-powered platform designed to help farmers with crop trading, price prediction, and microfinance support. It connects farmers directly with buyers and provides intelligent insights using machine learning.

---

## Features
- Direct farmer-to-buyer crop trading
- AI-based crop price prediction
- Smart crop recommendations
- Microfinance and loan support
- Secure login and authentication
- Transparent and user-friendly system

---

### Frontend

- MySQL

### AI & ML
- Scikit-learn
- TensorFlow

- Random Forest

## Advantages
- Reduces intermediaries
- Improves farmer profits
- Provides smart decision support
- Ensures secure and transparent transactions
